    Welcome to the Embedded and Real-Time Systems Course

    This course is taught by Prof. Kavi Arya and Prof. Krithi Ramamritham

    Course Outline

    Part 1

    1. Introduction to the ARM® Cortex™ M4F and Peripherals
    2. Introduction to Code Composer Studio and Energia
    3. Understanding Tiva Ware, Initialization, and GPIO
    4. Interrupts and the Timers
    5. PWM and Servo Motor Interfacing
    6. ADC and Interfacing Joystick
    7. UART and Interfacing LCD (16x2)
    8. USB
    9. Interfacing Sensors(LM35) and using the Hibernation Module

    Advanced Topics

    1. Memory and Security (EEPROM)
    2. SPI, I2C and IMU Interfacing (Optional: Synchronous Serial Interface) 3 The Floating Point Unit (Generating Sine wave and profiling)
    3. uDMA (Transfer large data to and from UART using uDMA)

    Part 2

    1. Basic concepts of Operating Systems
    2. Interfacing Graphics LCD
    3. LED blinking using HWI and SWI
    4. Experiments based tasks, mutex, semaphores, and locks

    Mini Project

    • Game development based on the concepts learned
